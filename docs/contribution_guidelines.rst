This page outlines how to contribute to the **Meetro** project.

-------------------

2. **Branching**:

3. **Code and Commit**:
   - Follow the coding guidelines below.

4. **Push and Submit a PR**:
   Push your changes and open a pull request (PR) on GitHub:
